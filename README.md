# DevToolkit - 开发者工具箱

一款面向开发者的 Chrome 浏览器扩展，提供常用的编码/解码、格式转换工具。

## 功能

- **JSON 格式化** — 格式化、压缩、校验 JSON
- **时间戳转换** — 秒/毫秒互转，日期↔时间戳
- **Base64** — 编码/解码
- **Hash** — MD5、SHA-1、SHA-256、SHA-512
- **颜色转换** — HEX ↔ RGB ↔ HSL，取色器
- **URL 编码** — encodeURIComponent / decodeURIComponent

## 安装

1. Chrome 地址栏输入 `chrome://extensions`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本文件夹
